# ESTHETIQUE Mobile

Expo Router mobile app for the ESTHETIQUE brand.

## What this app keeps from the web product

- Dark emerald industrial aesthetic
- Where style meets purpose. story
- Mobile auth with bearer tokens
- Shared API shapes for products, orders, plants, impact, and leads

## Local setup

1. Install dependencies from `mobile/package.json`
2. Set `EXPO_PUBLIC_API_URL` to the web backend
3. Run `npm run start` or `bunx expo start` inside `mobile/`
